## How To Run

### Run the backend

1. Install the requirements
```bash
python -m venv env
pip install -r requirements.txt
```
2. Run the server
```bash
cd backend
uvicorn api.main:app --reload
```

### Run the frontend

1. Install the packages
```bash
npm i
```
2. Run the app
```bash
npm run dev
```
